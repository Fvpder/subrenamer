# subRenamer (字幕批量重命名工具)

subRenamer 是一个用于批量重命名字幕文件的本地工具。它的主要作用是将字幕文件名一键重命名为与对应的视频文件名完全匹配的格式（附带 `.zh-cn` 后缀），方便大多数播放器自动挂载识别字幕。

提供了 **命令行 (CLI)** 与 **图形界面 (GUI)** 两个版本供不同使用习惯的用户选择。

## ✨ 核心特性

- **支持季数子目录**：视频和字幕主目录下**必须**包含同名的子文件夹（例如 `Season 1`, `Season 2`），程序会自动遍历并匹配。
- **智能排序匹配**：将视频文件和字幕文件分别按文件名升序排序后，进行一对一对应重命名。
- **独立防错机制**：如果某个季数文件夹中，**视频数量与字幕数量不一致**，程序会主动跳过该季，防止出现集数错位。
- **预览模式 (Dry Run)**：两个版本均默认提供预览模式，在真正执行重命名前可以仔细核对重命名计划。
- **多格式支持**：
  - 视频格式：`.mp4`, `.mkv`, `.avi`, `.ts`
  - 字幕格式：`.srt`, `.ass`, `.ssa`, `.sub`

## 📁 目录结构要求示例

在运行工具前，请确保您的视频目录和字幕目录都采用了相似的“季数子文件夹”结构：

```text
【视频主目录】
├── Season 1
│   ├── Show.S01E01.1080p.mp4
│   └── Show.S01E02.1080p.mkv
└── Season 2
    └── Show.S02E01.1080p.mp4

【字幕主目录】
├── Season 1
│   ├── sub_epi_1.srt
│   └── sub_epi_2.ass
└── Season 2
    └── ep1_sub.srt
```

## 🚀 快速开始

请确保您已安装 Python 3 环境。本工具仅使用了标准库，**无需安装任何额外依赖**。

### 方式一：图形界面版 (GUI) - ✨ 推荐

GUI 版本提供了友好的交互界面以及清晰的操作日志输出，适合日常大部分场景使用。

```bash
python rename_subs_GUI.py
```

**使用步骤**：
1. 点击“浏览...”按钮，分别选择【视频主文件夹】和【字幕主文件夹】。
2. 点击 **“🔍 预览重命名 (Dry Run)”**，在底部的极客风格日志区域中查看将会发生怎样的名称变更。
3. 确认日志中展示的重命名映射关系无误后，点击 **“🚀 开始执行重命名”**，完成真实的文件重命名。

### 方式二：命令行版 (CLI)

CLI 版本适合习惯终端操作的用户，或用于更复杂的自动化批量脚本中。

```bash
# 查看帮助和说明
python rename_subs_CLI.py -h

# 1. 预览模式（默认状态即为 Dry Run，不会实际更改文件）
python rename_subs_CLI.py "D:\Your\Video\Path" "D:\Your\Subtitle\Path"

# 2. 实际执行模式（须附加 --execute 参数才会真正重命名文件）
python rename_subs_CLI.py "D:\Your\Video\Path" "D:\Your\Subtitle\Path" --execute
```

## ⚠️ 注意事项

1. **集数对应**：由于程序是通过对文件**按名称字母顺序升序排序**来对应集数的，因此需要确保单个 Season 文件夹中视频和字幕的第一集、第二集……在排序后都处于相同的索引顺位。
2. **不可逆性**：真正执行改名操作后较难批量回退，**强烈建议首次使用时务必先使用预览模式（Dry Run）**观察日志输出中的文件匹配是否完美对应。
